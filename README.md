# Coordinates Generating GOPP Toolbox

Developed by Eng. Sarah ElSawy

## Description
A custom ArcGIS Pro toolbox for generating and managing coordinates with their north and west directions. It includes a tool to batch-number feature classes based on a specified grouping field, allowing efficient numbering of different coordinates according to their urban boundaries.

## Tools Included
- Generating Coordinates– Built with ArcGIS ModelBuilder
- Number FCs by Group Field – Python Script

## Requirements
- ArcGIS Pro (Individual or higher)

## How to Use
1. Open ArcGIS Pro
2. Add the toolbox (.atbx)
3. Open the Generate Coordinates tool (ModelBuilder).
4. Select the input boundary feature class.
5. Enter the name for the new coordinates feature class.
6. Select any fields you want to remove.
7. Run the Generate Coordinates tool.
8. Open the Number Feature Classes by Group Field tool (Python script).
9. Select the coordinates layer you just created.
10. Choose the grouping field
11. Run the Number Feature Classes by Group Field tool.
